 <h1 align="center" style="margin: 30px 0 30px;display: flex;align-items: center;font-weight: bold;">
 xjy_admin
 </h1>

一个基于Vue3+Element-plus+TypeScript搭建的轻量级（通用）后台管理系统模板，本模板相比其他后台管理模板，集成的插件和封装功能比较少，简单，通俗易懂。
